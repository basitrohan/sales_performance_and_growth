# Sales Performance & Growth Analysis Dashboard

A strategic overview of global revenue trends, product velocity, and forward-looking growth projections. This dashboard provides a comprehensive look at sales health across multiple teams, regions, and product categories.

## 📈 Key Performance Indicators (KPIs)

The dashboard monitors top-line metrics for organizational health:

* **Total Revenue:** 6M
* **Customer Base:** 177K
* **Volume Sold:** 380K
* **Efficiency Score:** 35.02

## 📊 Visualizations & Insights

1.  **Revenue by Quarter & Forecasting:** A time-series area chart tracking revenue from January 2021 through early 2022, including a forward-looking projection (gray shaded area) for strategic planning.
2.  **Revenue by Category:** A comparative bar chart breaking down sales into product types: **Bars ($3.0M)**, **Bites ($2.0M)**, and **Other ($1.2M)**.
3.  **Revenue by Region:** A geographical performance analysis showing market leadership in **India ($1.14M)**, followed by **New Zealand ($1.07M)**, the **UK**, and **Australia**.
4.  **Revenue by Size:** A donut chart illustrating the distribution between different product sizes (45.73% vs 54.27%).
5.  **Product Revenue vs. Volume (Scatter Plot):** A correlation analysis showing the relationship between revenue generated and boxes sold by individual product lines.

## 🤖 Data Generation
The data used to build this dashboard is **fully synthetic and AI-generated**.
* **Methodology:** Generated using a Python script to create realistic e-commerce and retail transaction logs.
* **Purpose:** This project serves as a portfolio demonstration to showcase data storytelling, forecasting implementation, and professional dashboard design while maintaining data privacy.

## 🛠️ Tech Stack
* **Visualization Tool:** Power BI
* **Data Processing:** Excel
* **Team Focus:** Comparative analysis for teams: **Delish, Jucies, Special, and Yummies**.
